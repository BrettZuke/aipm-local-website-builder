# `badges/` (trust badges — empty until Stage 10.1 populates)

The niche template's `niche-playbook/trust-signals.json` declares which
badges the template renders. Stage 10.1 copies the badge files referenced
by `brandDNA.trust_badges[].filename` into this folder, sourced from
`templates/{niche-slug}/niche-playbook/trust-badges/`.

This directory ships empty so no client's manufacturer/certification logos
ever end up in the public repo.
