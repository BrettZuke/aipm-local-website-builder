# Reference configs

Three filled `brand-dna.js` configs that show what a complete, well-built config
looks like, and how the variance engine makes each niche look different from the
one factory template. Copy one as a starting point, or read it to see how every
field is filled.

Each uses a different combination of the three layout dials (see
`../VARIANCE-ENGINE.md`):

| File | Niche | Blueprint | Hero | Vibe | Theme |
|---|---|---|---|---|---|
| `roofer.brand-dna.js` | Roofing | trust-first | split-form | signal | dark, copper |
| `garage.brand-dna.js` | Auto / MOT | showcase-first | full-bleed | structural | dark steel, red |
| `plumber.brand-dna.js` | Plumbing | trust-first | editorial-split | structural | light, blue |

To preview one locally: copy it over `../src/config/brand-dna.js`, drop matching
images into `../public/`, then run `npm install && npm run build && npm run preview`.

These are invented demo brands for reference only, not real clients.
