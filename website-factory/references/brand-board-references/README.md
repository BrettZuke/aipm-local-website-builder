# `brand-board-references/` (reference brand boards — empty by default)

Drop your own reference brand boards here as inspiration for Stage 8
(brand-board generation). Each reference should be:

- A single image (`.jpg`/`.png`/`.webp`)
- Named with the niche or theme it represents (e.g. `roofing-dark-bold.jpg`)
- 1280×800 or larger

Stage 8 reads this folder optionally; if empty, brand-board generation
falls back to the prompt-only mode driven by `brand-dna.json`.

The repo ships this folder empty. Nothing is committed here unless you
add your own references locally.
